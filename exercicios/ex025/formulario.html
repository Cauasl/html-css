Formulários

Todo formulário precisa estar dentro da tag <form>

Atributos form

O atributo "autocomplete" tem dois valores, e faz com que apareça ou não a sugestão para completar o que você está escrevendo. Off desliga o autocomplete, e On liga o autocomplete.

O atributo "action" faz com que quando o botão enviar seja apertado (que é o input submit) envie o formulário para o servidor. No caso seria para um servidor que utiliza php.

Atributos Input

O atributo "name" é mais útil para html e php, e "id" é mais útil para javascript e um pouco para html (No caso para o uso do label).

Input submit

Faz com que envie o formulário.

GET e POST

GET e POST são métodos (method) do formulário, o método GET (método padrão) exibi na url os dados que foram enviados ao formulário e o POST impede que os dados aparecam na url, mas isso não torna mais seguro, porque ainda é possível ver os dados que foram enviados. Isso é possível ver com clicando em especionar > network > headers, ali no cadastro é possível ver os dados enviados.

Uso dos métodos

O uso do GET é melhor em envio de noite informações que NÃO são sensíveis, como nome, sobrenome, peso e etc. E também para formulários que não ultrapassam três mil letra.

O uso do POST é para informações um pouco sensíveis, como senhas, número de cartão de crédito, envio de arquivos como fotos ou imagens, mas o ideal é aprender no futuro https para ser mais seguro.

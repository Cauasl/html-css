Conteúdo Principal

Para impedir que o site principal não seja afetado por um site possivelmente malicioso é bom utilizar o atributo 'sandbox'.

Abaixo tem dois iframes, um que pode coletar dados podendo ser malicioso e trazendo problema para o usuário e para quem criou e o outra que evita a coleta de dados podendo evitar de trazer problemas.

Sem 'sandbox'

Consegue enviar dados para o site dentro do iframe, causando possíveis problemas.

Com 'sandbox'

Não consegue enviar ou coletar dados, impedindo possíveis problemas.
